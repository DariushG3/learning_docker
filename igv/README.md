## README

[IGV Web App](https://github.com/igvteam/igv-webapp) as a [Docker image](https://hub.docker.com/r/dceoy/igv-webapp).

```bash
docker pull dceoy/igv-webapp
```

Run Docker.

```bash
docker run --rm \
           -p 8080:8080 \
           dceoy/igv-webapp
```

Now head to http://localhost:8080/ and that's it!
